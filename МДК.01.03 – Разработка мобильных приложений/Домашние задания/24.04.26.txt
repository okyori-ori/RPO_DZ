    'white': Colors.white,
    'red': Color.fromARGB(255, 192, 105, 99),
    'green': Color.fromARGB(255, 97, 204, 112),
    'blue': Color.fromARGB(255, 74, 127, 170),
  };
1. Что делает этот фрагмент?
Создаёт словарь для хранения соответствия между названиями цветов (строки) и их значениями (Color). Позволяет быстро получить цвет по его названию.

2. Почему удобнее использовать словарь (для разноцветных заметок)?
Быстрый доступ — получаем цвет по ключу за O(1)

Легко расширять — просто добавляем новую пару в словарь

Можно получить список всех цветов через .keys для отображения в UI

Одна строка кода вместо if/else или switch

3. Чем словарь отличается от списка?
Список (List)	Словарь (Map)
Доступ по индексу (числу)	Доступ по ключу (строка, число и т.д.)
list[0]	map['key']
Элементы могут повторяться	Ключи уникальны
Поиск медленный (перебор)	Поиск быстрый (по хэшу)
